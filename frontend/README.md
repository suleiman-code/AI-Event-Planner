# Frontend - AI Event Planner

React + Vite + TailwindCSS frontend for the event planning system.

## Setup

```powershell
npm install
```

## Run Development Server

```powershell
npm run dev
```

App runs at: http://localhost:3000

## Build for Production

```powershell
npm run build
```

## Preview Production Build

```powershell
npm run preview
```
